# Aplikasi Web Izin Staff

Aplikasi web untuk mengelola izin keluar staff dengan sistem kuota dan batasan waktu.

## Fitur

### 1. Sistem Login
- Login dengan nama staff dan password
- Hanya bisa login dalam jendela waktu 2 jam dari shift yang ditentukan
- Akses admin dengan password khusus

### 2. Kuota Izin
- 4x izin 15 menit per hari
- 3x izin makan 7 menit per hari
- Hanya 1 jobdesk yang dapat keluar izin dalam waktu bersamaan

### 3. Dashboard Staff
- Tampilan nama staff dan shift
- Informasi kuota izin
- Daftar staff yang sedang izin
- Riwayat izin pribadi
- Jam digital real-time

### 4. Panel Admin
- Kelola shift kerja
- Kelola daftar jobdesk
- Pengaturan jumlah izin
- Customisasi tampilan (logo, background, warna)
- Laporan dan monitoring
